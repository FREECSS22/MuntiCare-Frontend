# 🏥 MuntiCare – Smart Healthcare Management System

MuntiCare is a **Smart Healthcare Management System** designed to assist the **Muntinlupa City Health Office** in improving healthcare service accessibility, patient management, and appointment scheduling.

The system integrates an **AI-powered assistant** that helps users navigate healthcare services, answer common health-related questions, and guide patients through appointment processes.

This project was originally developed as a **full-stack web system using Laravel with a backend and database**. For portfolio purposes, a **demo version is deployed on Vercel**, showcasing the interface and core features using **local storage for temporary data handling**.

---

# 📌 Features

## 🧠 MuntiCare AI Assistant

The system includes an AI-powered chatbot designed to assist patients and users.

### Symptom Checker
Provides general health guidance based on symptoms entered by the user.  
The assistant suggests possible next steps and encourages users to consult healthcare professionals when necessary.

### Appointment Assistant
Helps users:
- Book appointments
- View clinic schedules
- Get information about appointment requirements
- Navigate appointment-related processes

### Health FAQ
Answers common questions about:
- Clinic services
- Operating hours
- Health programs
- Requirements for consultations

### Medicine Guidance
Provides general information about medicines such as:
- Usage reminders
- Safety precautions
- Basic medicine information

> ⚠️ The AI assistant **does not provide medical diagnoses** and encourages users to consult healthcare professionals for proper medical advice.
